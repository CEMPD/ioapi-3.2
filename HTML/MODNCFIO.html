Fortran-90 MODULE MODNCFIO

New for I/O API-3.2!! Developed from (and extends) I/O API/netCDF/PnetCDF INCLUDE-files NETCDF.EXT and pnetcdf.inc,

This module is purely a "declarations-module", which is used to resolve various incompatibilities between different versions of netCDF and (if PnetCDF/MPI distributed I/O is enabled)_PnetCDF INCLUDE-files. The options are controlled by preprocessor symbols at libioapi.a build-time:

  • If IOAPI_PNCF is defined (as it is in Makefile.pncf), then PnetCDF/MPI distributed I/O is enabled.

  • If IOAPI_NCF4 is defined (it is commented-out in the Makefile.*), then the new NetCDF-4 features are enabled (including 64-bit-INTEGER variables and attributes, and HDF file-formats).

  • Otherwise, only the "traditional" netCDF-3 routines and parameters are turned on. 64-bit-INTEGER operations will return failure, and write an appropriate error message.
This of course leads to four (link-incompatible!) distinct netCDF-version related versions of the I/O API library (and PVM-enabled or not option, as distinguished by preprocessor symbol IOAPICPL defined in Makefile.cpl turns it into a total of 8 versions in all).

It is also hoped that the source code for this module is somewhat more readable than the original INCLUDE-files :-)
